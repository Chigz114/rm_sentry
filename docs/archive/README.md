# Archive

This directory records deprecated or non-default project paths so they do not compete with the active pipeline.

Archive docs do not mean the source files are always deleted. They mean the path is not a default debugging or implementation target.

Rules:

- Keep archive notes factual.
- Do not restore archived paths without an explicit task.
- If an archived implementation is still present in source, point to the active replacement.
- If an archived implementation has been deleted, point to Git history or the replacement.
