# History

This directory preserves confirmed historical reasons, decisions, and bug records.

Rules:

- Do not invent or backfill uncertain history.
- Prefer a blank template over a confident but unverifiable record.
- Use `timeline.md` for chronological facts only.
- Use `index_by_module.md` and `index_by_symptom.md` as lookup indexes.
- Use `decisions/` for durable choices and rejected alternatives.
- Use `bugs/` for root-cause records with evidence or a clear source.

The long historical work log remains at `docs/height_gated_traversability_plan.md`.
